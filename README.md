# 🌿 Greencore

> Jogo educativo desenvolvido por alunos da **Universidade Federal Rural de Pernambuco (UFRPE)** para fins didáticos.
>
> **Autores:** Lucas Gabriel Bomfim Rodrigues / Eloísa Fernanda Pimentel

---

## 📖 Sobre o Jogo

Greencore é um jogo de quiz temático sobre **meio ambiente e sustentabilidade**. O jogador responde perguntas de múltipla escolha sobre temas como reciclagem, pegada ecológica, energia renovável e conservação da natureza, acompanhado por personagens ilustrados (arara-azul, tartaruga sábia, robô ecológico, entre outros). Há quatro níveis de dificuldade — Fácil, Médio, Difícil e Pesadelo — cada um com tempo limite e pontuação diferentes. Ao acertar todas as perguntas de um nível, o jogador conquista um troféu exibido na Sala de Troféus.

---

## ✅ Requisitos

- **Python 3.10+**
- **Tkinter** — normalmente já vem com o Python; caso não esteja instalado:
```bash
  sudo apt install python3-tk
```
- **Pillow** — biblioteca para carregamento das imagens:
```bash
  pip install Pillow
```

---

## ▶️ Como Rodar

```bash
# Clone ou extraia o projeto e entre na pasta
cd "Greencore - jogo Original"

# Execute o jogo
python main.py
```
